💳 Credit Card Default Prediction

This project is a machine learning model that predicts whether a credit card customer will default on their payment next month based on historical financial and demographic data.

🎯 Goal

Build a classification model to identify high-risk customers and help reduce credit risk for financial institutions.

📊 Dataset Features

Includes customer information such as:

Demographics (age, gender, education, marital status)
Credit limit
Payment history
Bill and payment amounts

Target:

default_payment_next_month (0 = No Default, 1 = Default)
🤖 Models Used
Logistic Regression
Decision Tree
Random Forest
XGBoost
📈 Evaluation
Accuracy
Precision
Recall
F1-score
ROC-AUC
🛠️ Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn# credit-card-default-prediction
This project focuses on building a machine learning model to predict whether a credit card customer is likely to default on their payment in the next billing cycle. The goal is to help financial institutions assess credit risk and make data-driven lending decisions.
